# Yuanzhen Zhao — Personal Website

A bilingual static portfolio focused on cross-disciplinary problem solving across physics, statistics, generative modelling, scientific computing, and markets.

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run validate:content
npm run typecheck
npm run lint
npm run build
```

The static export is written to `out/`. English is served at `/` and Chinese at `/zh/`.
